# Discord Status Rotator

This Python script rotates your Discord profile's custom status. It uses the Discord API to change your account status at regular intervals based on the configuration in `config.json`.

## Clone the Repository

## 1 To clone this repository to your local machine, use the following command:
      git clone https://github.com/NullBytesx/Simple-Status-Rotator

## 2 Requeriments
      Python 3.6 or higher
      requests package
      colorama package

## 3 Usage 
      cd Simple-Status-Rotator
      python main.py
